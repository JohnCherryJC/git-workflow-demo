# Git Workflow Demo Project

This project demonstrates correct usage of Git:
- branching
- semantic versioning
- changelog
- tags
- labels
- merge process

## What the application does
The application is a demo project for learning Git workflows.

## For developers
The project uses feature branches and hotfix branches.
All changes are merged into master via pull requests.

## Tasks order
1. Project initialization
2. Feature development
3. Bug fixing
4. Version release

## Features
- Git branching
- Semantic versioning
- Changelog support
